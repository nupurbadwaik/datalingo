# DataLingo
DataLingo allows users to query databases using simple English instead of complex SQL. It uses the Mistral 7B AI model to convert natural language prompts into precise database queries. Built with Spring Boot (backend) and ReactJS (frontend), the system supports multiple databases, query validation, and result visualization. Targeted at business analysts, managers, and non-technical users, it reduces technical barriers, boosts productivity, and democratizes data access. Combining NLP, database management, and secure APIs, DataLingo showcases practical application of AI and software engineering principles to solve real-world data accessibility challenges.
